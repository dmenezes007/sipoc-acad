# Security Policy

## Reportando Vulnerabilidades de Segurança

A segurança do SIPOC ACAD é levada a sério. Se você descobrir uma vulnerabilidade de segurança, por favor **NÃO** abra uma issue pública.

### Como Reportar

Envie um e-mail para: **academia@inpi.gov.br**

Inclua:
- Descrição detalhada da vulnerabilidade
- Passos para reproduzir o problema
- Possível impacto da vulnerabilidade
- Sugestões de correção (se houver)

### O que esperar

1. **Confirmação**: Confirmaremos o recebimento em até 48 horas
2. **Análise**: Avaliaremos a vulnerabilidade em até 7 dias
3. **Correção**: Trabalharemos em uma correção o mais rápido possível
4. **Divulgação**: Coordenaremos a divulgação pública após a correção

## Versões Suportadas

| Versão | Suportada          |
| ------ | ------------------ |
| 1.x.x  | :white_check_mark: |

## Políticas de Segurança

- Mantenha dependências atualizadas
- Não commite secrets ou credenciais
- Use HTTPS para todas as comunicações
- Siga as melhores práticas de segurança web

## Agradecimentos

Agradecemos aos pesquisadores de segurança que reportam vulnerabilidades de forma responsável.
